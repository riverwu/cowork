# 客户运营经营结构复盘

受众：COO 经营例会。PPT 需要解释客户运营体系的组织分工、关键经营指标，以及成熟度模型如何落到执行优先级。

## 公司结构与经营数据

- 运营委员会负责经营节奏、周经营会和跨部门资源协调。
- 运营委员会本月核心指标：续费率 94%，净收入留存 NRR 108%。
- 客户成功部负责续约、扩容和客户健康管理；团队 32 人，续约金额 8,420 万。
- 交付服务部负责上线、里程碑和 SLA；平均上线周期 18 天，SLA 达成率 92%。
- 产品运营部负责活跃、工单闭环和数据看板；月活跃率 71%，客户健康分 76。
- 客户成功部下设重点客户组和中小客户组。
- 交付服务部下设实施 PMO。
- 产品运营部下设数据分析组。

## 经营成熟度金字塔

从上到下：
1. 北极星指标：净收入留存 / 续费率，metric 108%，badge “目标94%”。
2. 增长抓手：重点客户扩容、风险客户挽回、低触达运营，metric 3条，badge “策略”。
3. 运营机制：周会 / 预警 / 闭环，metric 6项，badge “机制”。
4. 执行底座：健康分76、闭环率91%、准时率92%，metric 12个，badge “数据底座”。

金字塔需要明显体现层宽和层高变化；不能看起来像四个完全相同的卡片。
