# 物理力学提纲

> 从牛顿三大定律出发，理解力学的完整逻辑体系——为什么是这样，它们之间如何关联。

---

## 一、核心基础：牛顿三大定律

### 1.1 第一定律（惯性定律）

> 物体在合外力为零时，保持静止或匀速直线运动。

**意义**：定义了"不受力"的理想状态是自然界的默认状态，运动的改变必须有原因——这为"力"的概念提供了存在依据。惯性是所有物体固有的属性，质量越大，惯性越大。

**数学表达**：若 $\vec{F}_{net} = 0$，则 $\vec{v} = \text{const}$

---

### 1.2 第二定律（加速度定律）

> 合外力等于动量随时间的变化率。

$$\vec{F}_{net} = \frac{d\vec{p}}{dt} = m\vec{a} \quad (\text{当 } m = \text{const})$$

**为什么是 $F=ma$，而不是 $F=mv$ 或 $F∝v$？**

- **历史动机**：伽利略通过斜面实验发现，速度本身不能衡量力的大小——因为物体在不受力时速度可以很大（匀速）。真正需要解释的是"速度为什么会变"，即加速度。
- **逻辑检验**：若 $F∝v$，则静止物体 ($v=0$) 受到恒力时仍不动——这与日常经验矛盾。若 $F∝a$，则恒力产生恒定加速度，自由落体越落越快，与实验一致。
- **量纲一致**：$F=ma$ 使力的单位（牛顿）定义为"使 1kg 获得 1m/s² 加速度的力"，可操作、可测量。
- **可证伪性**：直接推出 $a=F/m$，可被实验精确检验。

**质量的意义**：质量在这里出现两次——**惯性质量**（惯性定律中"抗拒速度改变"的属性）和**引力质量**（万有引力中的"产生引力"的属性）。它们在实验中精确相等，这是广义相对论的出发点，但牛顿力学中视为巧合。

**适用范围**：仅在惯性参考系中成立。非惯性系中需引入惯性力。

---

### 1.3 第三定律（作用与反作用定律）

> 两物体间的作用力与反作用力大小相等、方向相反、作用于同一直线。

$$\vec{F}_{AB} = -\vec{F}_{BA}$$

**意义**：这是整个牛顿力学体系的"内部自洽性"保证。若无第三定律，则系统内力无法定义，动量守恒无法推导，力的测量也无从谈起——因为你测到的"力"无法区分主动施加还是被动反作用。

**常见误解**：作用力与反作用力**不是**一对平衡力（平衡力是作用于同一物体，大小相等方向相反）。

---

## 二、冲量与动量定理

### 2.1 冲量的定义

**冲量**是力在时间上的累积效应：

$$\vec{I} = \int_{t_1}^{t_2} \vec{F}\,dt$$

- 若力为恒力：$\vec{I} = \vec{F}\Delta t$
- 若力为变力：$\vec{I} = \vec{F}_{avg}\Delta t$

**物理意义**：冲量是"力传递了多少运动量"的量度。它是一个**过程量**（与时间历程有关），而非状态量。

**量纲**：$[\vec{I}] = N·s = kg·m/s$，与动量完全相同——这暗示了它们之间的深层联系。

---

### 2.2 动量定理（冲量-动量定理）

将牛顿第二定律两边对时间积分：

$$\vec{F}_{net} = \frac{d\vec{p}}{dt} \implies \int_{t_1}^{t_2}\vec{F}_{net}\,dt = \Delta\vec{p}$$

即：

$$\boxed{\vec{I} = \vec{F}_{avg}\Delta t = \Delta\vec{p} = \vec{p}_2 - \vec{p}_1}$$

**文字表述**：合外力的冲量等于物体动量的变化。

**这是牛顿第二定律的积分形式**——第二定律告诉你"瞬时关系"（力→加速度），动量定理告诉你"全过程关系"（冲量→动量变化）。两者完全等价，在不同场景下各有用处。

---

### 2.3 动量定理 vs 动能定理

| | 动量定理 | 动能定理 |
|---|---|---|
| **表达式** | $\vec{F}\Delta t = \Delta\vec{p}$ | $\int\vec{F}\cdot d\vec{r} = \Delta E_k$ |
| **核心物理量** | 矢量：方向敏感 | 标量：方向无关 |
| **揭示的信息** | 速度的方向变化 | 速度的大小变化 |
| **内力是否做功** | 不影响总动量（第三定律保证） | 可能影响总动能（碰撞损失） |

**关键洞察**：动量是矢量，动能是标量。这意味着：

- 动量定理可以处理方向突变（如绳子拉物体改变运动方向）而无需分析轨迹
- 动能定理处理的是"做了多少功"，与路径有关
- 两者相互独立，不可相互推导

---

### 2.4 动量定理的典型应用

**（1）缓冲与防护的原理**

安全气囊、气垫床、蹦极绳——它们的共同原理是：增大 $\Delta t$ 以减小平均冲击力 $\vec{F}_{avg} = \frac{\Delta\vec{p}}{\Delta t}$。同样的动量变化，时间拉得越长，力越小。

**（2）流体冲击问题**

$$\vec{F}_{avg} = \frac{\Delta m}{\Delta t}\Delta\vec{v} = \dot{m}\,\vec{v}_{out}$$

典型应用：火箭推进原理、消防水枪反作用力、水轮机叶片受力分析。

**（3）二维/三维分解**

动量定理是矢量方程，可以在 $x,y,z$ 方向上分别列方程。复杂碰撞问题中，这比能量守恒（标量）更有用——因为能量守恒要求碰撞是弹性的，而动量守恒对碰撞类型无要求（只要合外力为零）。

## 三、动力学核心定理：从牛顿定律推导出的四大守恒

### 3.1 动量守恒

**推导**：

由 $F=ma$ 和 $a = \frac{dv}{dt}$：

$$\vec{F}_{net} = \frac{d(m\vec{v})}{dt} = \frac{d\vec{p}}{dt}$$

若 $\vec{F}_{net} = 0$（合外力为零），则：

$$\frac{d\vec{p}}{dt} = 0 \implies \vec{p} = \text{const}$$

**物理图像**：动量是"运动的量"的度量。合外力为零意味着系统与外界无动量交换，因此"运动的量"守恒。

**适用条件**：合外力为零。内力不影响系统总动量（由第三定律保证）。

---

### 3.2 能量守恒

**动能定理（工作-动能定理）**：

$$W = \Delta E_k = E_{k2} - E_{k1} = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$$

**推导**：

$$W = \int \vec{F}\cdot d\vec{r} = \int m\frac{d\vec{v}}{dt}\cdot \vec{v}\,dt = m\int v\,dv = \frac{1}{2}mv^2$$

**为什么有 $1/2$ 的系数？**

因为速度是连续变化的。"力做功使速度从 0 到 $v$"意味着平均"加速能力"是 $\frac{1}{2}F$（初始慢时做功效率低，后来快时同样力做功效率高）。积分 $\int_0^v mv\,dv = \frac{1}{2}mv^2$ 自然产生这个系数。

**势能与保守力**：

若力做功与路径无关（保守力），则存在势能函数 $U(\vec{r})$：

$$W = -\Delta U \implies F = -\nabla U$$

**势能函数一览**：

| 势能类型 | 表达式 | 公式 |
|----------|--------|------|
| 重力势能（地面附近） | $U = mgh$ | $F = mg$ |
| 引力势能（万有引力） | $U = -\frac{GMm}{r}$ | $F = \frac{GMm}{r^2}$ |
| 弹力势能 | $U = \frac{1}{2}kx^2$ | $F = -kx$ |
| 电势能 | $U = \frac{kq_1q_2}{r}$ | $F = \frac{kq_1q_2}{r^2}$ |

**机械能守恒**（只有保守内力做功时）：

$$E_{k1} + U_1 = E_{k2} + U_2 = \text{const}$$

---

### 3.3 角动量守恒

**定义**：$\vec{L} = \vec{r} \times \vec{p} = \vec{r} \times m\vec{v}$

**物理意义**：角动量是"绕某点旋转之量"的度量。类似于动量是"直线运动之量"。

**力矩与角动量变化率**：

$$\vec{\tau} = \frac{d\vec{L}}{dt} \quad \text{（力矩 = 角动量变化率）}$$

推导：$\vec{L} = \vec{r} \times m\vec{v}$，对时间求导：

$$\frac{d\vec{L}}{dt} = \frac{d\vec{r}}{dt}\times m\vec{v} + \vec{r} \times m\frac{d\vec{v}}{dt} = 0 + \vec{r}\times\vec{F} = \vec{\tau}$$

（因为 $\frac{d\vec{r}}{dt} = \vec{v}$ 与 $\vec{v}$ 平行，叉乘为零）

**守恒条件**：合外力矩为零时，$\vec{L} = \text{const}$。

**典型应用**：行星运动（太阳对行星的引力始终指向太阳，力矩为零）→ 角动量守恒 → 开普勒第二定律（等面积定律）。

---

### 3.4 四大守恒定律的横向关系

```
牛顿第二定律 (F = ma)
        ↓ （对时间积分）
    动量定理 (F·Δt = Δp)
        ↓ （合外力为零）
    动量守恒 (p = const)
        ↓ （力是保守力，路径无关）
    动能定理 + 势能定义
        ↓ （机械能 = 动能 + 势能）
    机械能守恒 (E = const)
        ↓ （对位置积分，力矩 = r×F）
    角动量定理 (τ = dL/dt)
        ↓ （合外力矩为零）
    角动量守恒 (L = const)
```

**关键洞察**：四大守恒定律并非彼此独立——它们全部由牛顿第二定律在不同维度（时间/空间/转动）上的积分或变换推导而来。守恒定律是牛顿定律的"积分形式"，揭示的是更深层的对称性：

| 守恒定律 | 对应的自然对称性 |
|----------|-----------------|
| 动量守恒 | 空间平移对称性（物理定律不因地点而变） |
| 能量守恒 | 时间平移对称性（物理定律不因时刻而变） |
| 角动量守恒 | 空间旋转对称性（物理定律不因方向而变） |

> 这是诺特定理（Noether's Theorem）的核心：每一条守恒定律背后，都有一条自然对称性。

## 四、万有引力：从开普勒到牛顿

### 4.1 开普勒三定律（经验定律，1609-1619）

| 定律 | 内容 | 几何含义 |
|------|------|----------|
| 第一定律 | 行星绕太阳做椭圆运动，太阳在焦点 | 否定了正圆轨道的千年执念 |
| 第二定律 | 相同时间内，行星与太阳连线扫过相同面积 | $\frac{dA}{dt} = \text{const}$ |
| 第三定律 | $T^2 ∝ r^3$（轨道周期的平方与半长轴的立方成正比） | 所有行星共享同一常量 |

### 4.2 牛顿的万有引力定律

$$F = G\frac{Mm}{r^2}$$

**为什么是 $1/r^2$，而不是 $1/r$ 或 $1/r^3$？**

- **理论动机**：若太阳对行星的力与距离成正比（$F∝r$），则行星会在椭圆焦点上做简谐运动，这与开普勒观测的椭圆轨道不一致。
- **数学推导（牛顿的证明）**：
  1. 由开普勒第二定律（等面积）→ 角动量守恒 → 引力指向太阳（力矩为零）
  2. 由开普勒第三定律（$T^2∝r^3$）+ 向心力公式 $F_c = m\frac{4\pi^2 r}{T^2}$ → $F_c ∝ \frac{m}{r^2}$（与行星质量 $m$ 成正比，与 $r^2$ 成反比）
  3. 由第三定律（作用反作用）→ 太阳也受行星的力，大小相等 → $F ∝ M$（太阳质量）

**引力与重力的区别**：
- 重力 $mg$ 是地球对地面物体的引力（近似，因为地球自转有微小离心效应）
- 万有引力 $F = G\frac{Mm}{r^2}$ 是普遍适用的天体间相互作用

**为什么 $G$ 这么难测？**

因为引力是宇宙四力中最弱的。两个人的引力约为 $10^{-7}N$，需要极精密仪器。卡文迪许扭秤实验（1798）利用"光杠杆"放大微小扭转，间接测量了 $G$。

## 五、向心力与圆周运动

### 5.1 向心力的概念澄清

**向心力不是一种新类型的力，而是合外力在特定条件下的角色。**

在圆周运动中，必须有一个真实的力（重力、绳子拉力、摩擦力、万有引力……）充当向心力，使得：

$$F_{net} = \frac{mv^2}{r} \quad \text{指向圆心}$$

**常见误区**："离心力"——这是**假力**，只存在于非惯性参考系（转动参考系）中。在惯性系中分析圆周运动时，离心力根本不存在，不需要引入。

### 5.2 向心力的来源分析

| 情境 | 向心力的来源 |
|------|-------------|
| 圆锥摆 | 绳子拉力的水平分量 |
| 汽车转弯 | 静摩擦力（指向弯道内侧） |
| 地球公转 | 万有引力 |
| 卫星绕地球 | 万有引力 |
| 电子绕核旋转 | 库仑静电力 |

**注意**：如果合外力大小**不等于** $mv^2/r$，物体就不会做匀速圆周运动，而是做椭圆运动或其他曲线运动（如行星轨道）。

### 5.3 竖直平面内的圆周运动

**关键差异**：最高点和最低点的向心力需求不同。

- **最高点**（速度最小时，向心力需求最小）：$mg + T_{min} = \frac{mv^2}{r}$ → $T_{min} = \frac{mv^2}{r} - mg$
- **最低点**（速度最大，向心力需求最大）：$T_{max} - mg = \frac{mv^2}{r}$ → $T_{max} = \frac{mv^2}{r} + mg$

**过顶点的临界条件**：在最高点绳子刚好"松弛"（$T_{min}=0$）时，$v_{临界} = \sqrt{gr}$。这是**物理摆**（如过山车）能够完成圆周运动所需的最小速度。

### 5.4 圆周运动与一般曲线运动的关系

圆周运动是一般曲线运动的**局部近似**：在轨迹的极小区间内，将曲线视为圆弧，用当地曲率半径 $\rho$ 替代 $r$：

$$a_n = \frac{v^2}{\rho} \quad \text{（法向加速度）}$$

沿切线方向：$a_t = \frac{dv}{dt}$（切向加速度，改变速度大小）

## 六、牛顿力学在行星运动中的应用

### 6.1 从开普勒定律推导引力公式

这是牛顿力学最经典的成就之一——用万有引力解释了开普勒三定律。

**第一步：引力指向太阳（由开普勒第二定律推出）**

开普勒第二定律（等面积定律）：$\frac{dA}{dt} = \frac{1}{2}r^2\dot{\theta} = \text{const}$

对时间求导：$r^2\dot{\theta} = \text{const} = h$（常数）

这正是角动量守恒：$L = mrv_\theta = mr^2\dot{\theta} = mh$

由角动量守恒条件 $\tau = \vec{r}\times\vec{F} = 0$ → $\vec{F} \parallel \vec{r}$ → **引力指向太阳**。

**第二步：引力与 $1/r^2$ 成正比（由开普勒第二定律的几何分析推出）**

将极坐标下的运动方程分解为径向和切向分量。径向分量方程为：

$$m(\ddot{r} - r\dot{\theta}^2) = F_r(r)$$

利用 $h = r^2\dot{\theta}$，消去 $\dot{\theta}$：

$$m\left(\ddot{r} - \frac{h^2}{r^3}\right) = F_r(r)$$

由开普勒第一定律，行星轨道为椭圆 $\frac{p}{r} = 1 + e\cos\theta$（$p, e$ 为常数）。将这个几何形状代入运动方程，可以反推出 $F_r(r) = -\frac{k}{r^2}$（$k$ 为常数）。负号表示引力方向指向太阳。

**第三步：引力与行星质量成正比（由第三定律推出）**

若行星质量加倍，在相同轨道运动意味着 $F$ 必须加倍（因为 $a = v^2/r$ 不变，由 $F=ma$，故 $F$ 必须加倍）。因此 $F ∝ m$。

**第四步：引力与太阳质量成正比（由作用反作用定律）**

太阳对行星的引力 $F = G\frac{Mm}{r^2}$，由牛顿第三定律，行星对太阳的力大小相等，故也 $∝ M$。

**最终结论**：

$$F = G\frac{Mm}{r^2}$$

---

### 6.2 轨道方程与能量关系

**轨道方程（极坐标形式）**：

由能量和角动量守恒，行星轨道满足：

$$\frac{d^2u}{d\theta^2} + u = \frac{GM}{h^2}, \quad \text{其中 } u = \frac{1}{r}$$

解为圆锥曲线：$r = \frac{p}{1 + e\cos\theta}$

| 离心率 $e$ | 轨道类型 |
|-----------|----------|
| $e = 0$ | 圆轨道 |
| $0 < e < 1$ | 椭圆轨道（行星） |
| $e = 1$ | 抛物线轨道（逃逸临界） |
| $e > 1$ | 双曲线轨道（完全逃逸） |

**轨道机械能**：

$$E = \frac{1}{2}mv^2 - \frac{GMm}{r}$$

| 能量 $E$ | 轨道类型 |
|---------|----------|
| $E < 0$ | 椭圆轨道（束缚态） |
| $E = 0$ | 抛物线轨道（逃逸临界） |
| $E > 0$ | 双曲线轨道（逃逸后仍有余速） |

**关键洞察**：负能量意味着行星被太阳引力束缚，无法自行逃离。这与地球被太阳"捕获"在椭圆轨道中是完全一致的。

---

### 6.3 宇宙速度

| 速度 | 条件 | 公式 | 说明 |
|------|------|------|------|
| **第一宇宙速度** $v_1$ | 近地圆轨道 | $v_1 = \sqrt{gr} ≈ 7.9$ km/s | 刚好绕地球做圆周运动 |
| **第二宇宙速度** $v_2$（逃逸速度） | 抛物线轨道（$E=0$） | $v_2 = \sqrt{\frac{2GM}{R}} = \sqrt{2gr} ≈ 11.2$ km/s | 刚好能逃离地球引力 |
| **第三宇宙速度** $v_3$ | 逃离地球后还能逃离太阳 | $v_3 ≈ 16.7$ km/s | 从地球表面发射 |

**为什么 $v_2 = \sqrt{2}v_1$（正好是两倍根号）？**

因为引力势能与 $r$ 成反比（$U = -\frac{GMm}{r}$），而圆轨道动能是势能绝对值的 $-\frac{1}{2}$。逃逸需要将势能全部转化为动能（$E=0$），所以逃逸速度是圆轨道速度的 $\sqrt{2}$ 倍。

### 6.4 双星系统

两颗星绕共同质心运动，满足：

$$m_1r_1 = m_2r_2, \quad \vec{F}_{12} = -\vec{F}_{21}$$

万有引力同时提供两颗星的向心力：

$$G\frac{m_1m_2}{r^2} = m_1\omega^2 r_1 = m_2\omega^2 r_2$$

**开普勒第三定律的修正**（适用于双星系统）：

$$T^2 = \frac{4\pi^2}{G(m_1+m_2)}a^3$$

其中 $a = r_1 + r_2$ 是两颗星之间的距离。这正是通过观测双星运动推断恒星质量的方法。

**黑洞间接发现的原理**：通过观测双星系统中可见星的轨道参数，利用上述公式计算出不可见伴星的质量——若质量超过奥本海默极限（$≈3M_\odot$），则极可能是黑洞。

## 七、振动与波动：周期运动的核心规律

### 7.1 简谐运动（SHM）

**定义**：物体受回复力 $F = -kx$ 驱动而产生的运动。

**标准方程**：

$$m\frac{d^2x}{dt^2} = -kx \implies x(t) = A\cos(\omega t + \phi)$$

其中 $\omega = \sqrt{\frac{k}{m}}$，周期 $T = \frac{2\pi}{\omega}$。

**能量**：

$$E = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2A^2 = \frac{1}{2}mv_{max}^2 + \frac{1}{2}kx_{max}^2$$

**为什么弹簧振子有精确的正弦解？**

因为 $F=-kx$ 是**线性回复力**——力与位移成正比且方向相反。线性微分方程的解必然是三角函数。非线性回复力（如单摆大角度时 $F=-mg\sin\theta$）则没有简单解析解。

**单摆的近似**：

小角度时 $\sin\theta ≈ \theta$，方程变为 $\frac{d^2\theta}{dt^2} + \frac{g}{L}\theta = 0$，同样是简谐运动，周期 $T = 2\pi\sqrt{\frac{L}{g}}$（与振幅无关！）。大角度时周期变长，这是非线性效应的体现。

### 7.2 阻尼振动与共振

**阻尼振动方程**：

$$m\frac{d^2x}{dt^2} + b\frac{dx}{dt} + kx = 0$$

- **欠阻尼**（$b < 2\sqrt{km}$）：振幅指数衰减，频率略低于固有频率
- **过阻尼**（$b > 2\sqrt{km}$）：缓慢回到平衡，不振荡
- **临界阻尼**（$b = 2\sqrt{km}$）：最快回到平衡而不超调

**共振**：

受迫振动的振幅在 $\omega = \omega_{res} = \sqrt{\omega_0^2 - 2\beta^2}$（$\beta=b/2m$）处最大。共振现象说明：外界周期性驱动力与系统固有频率匹配时，能量输入效率最高——这解释了音叉的选取、桥梁的坍塌（塔科马海峡吊桥，1940）、以及核磁共振成像（MRI）的原理。

### 7.3 波动的基本方程

**波速**：

| 波的类型 | 介质条件 | 波速公式 |
|----------|----------|----------|
| 纵波（声波） | 气体/液体 | $v = \sqrt{\frac{B}{\rho}}$ |
| 横波（弦） | 细弦 | $v = \sqrt{\frac{T}{\mu}}$ |
| 电磁波 | 真空 | $v = c = \frac{1}{\sqrt{\varepsilon_0\mu_0}}$ |

**波长、频率、波速的关系**：

$$v = \lambda f = \frac{\lambda}{T}$$

**多普勒效应**（波源或观察者运动时观测频率变化）：

$$f' = f\frac{v \pm v_{observer}}{v \mp v_{source}}$$

（靠近取加号，远离取减号）

## 八、守恒定律的应用框架

### 8.1 碰撞问题：动量与动能的取舍

| 碰撞类型 | 动量 | 动能 | 特点 |
|----------|------|------|------|
| **完全弹性碰撞** | 守恒 | 守恒 | 理想情况，宏观极少（如低弹性介质中的硬球） |
| **完全非弹性碰撞** | 守恒 | **不守恒**（有内能损失） | 碰撞后物体粘连，如车祸黏车 |
| **非弹性碰撞** | 守恒 | 部分损失 | 介于两者之间 |

**为什么弹性碰撞中动能也守恒，而一般碰撞不守恒？**

碰撞过程中可能有永久形变（弹簧势能转化为热能）、声音辐射等能量耗散。动能守恒要求所有微观相互作用都是保守的（无摩擦、无形变），这在宏观世界极为罕见。

### 8.2 能量-动量联合问题

处理变质量问题（如火箭燃料喷出）时，**动量守恒通常比牛顿第二定律更可靠**，因为 $F=ma$ 在质量变化时形式复杂（需用动量定理 $F = \frac{dp}{dt}$）。

**典型例题框架**：
1. 分析系统，确定哪些力是外力、哪些是内力
2. 选守恒定律（能量？动量？角动量？）
3. 确定初末状态
4. 代入求解

| 公式 | 条件 |
|------|------|
| $\vec{p}_1 + \vec{p}_2 = \vec{p}_1' + \vec{p}_2'$ | 合外力为零 |

| $\vec{L} = \vec{r}\times m\vec{v} = \text{const}$ | 合外力矩为零 |

### 向心运动

| 公式 | 说明 |

## 十、学习建议：从定律到理解

1. **从实验出发**：每个定律最初都是对实验现象的归纳。理解"为什么是这个公式"，必须回到最初的实验动机。
2. **注意适用条件**：牛顿力学不适用于高速（需相对论）或微观（需量子力学）。每个公式都有边界。
3. **守恒定律优先**：面对复杂问题，先判断有哪些守恒量，往往比逐帧分析运动更高效。
4. **量纲分析检验**：所有物理公式左右量纲必须一致，这是检验正确性的第一道关卡。
5. **从特殊到一般**：先掌握匀加速直线运动（最简单的加速），再扩展到抛体、圆周、振动——每一步都有逻辑递进。
6. **区分"什么是力"与"力的角色"**：向心力不是一种新力，而是万有引力、绳子拉力等力在圆周运动中的"角色"。找向心力，就是找合外力中指向圆心的那个分量。
